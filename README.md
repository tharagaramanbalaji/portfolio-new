# Portfolio | Tharagaraman Balaji

A modern, high-performance portfolio website migrated from static HTML/CSS to **React + Vite**. It features a minimalist aesthetic with custom cursor effects, smooth transitions using Framer Motion, and a responsive design.

## Features

- **React SPA**: Smooth, lightning-fast navigation with React Router.
- **Framer Motion**: Premium, staggered animations and transitions.
- **Custom Cursor**: A dynamic, interactive cursor rebuilt in React.
- **Keyboard Navigation**: Quick access shortcuts for core sections (Home, Projects, Experience).
- **Vite-Powered**: Highly optimized build process and lightning-fast HMR.

## Tech Stack

- **React** (v19)
- **Vite**
- **Framer Motion**
- **Lucide React** (for UI icons)
- **Font Awesome** (for brand icons)
- **CSS3** (Custom variables and layout)

## Getting Started

### Prerequisites

- Node.js (v20.19+ or v22.12+)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

### Deployment

The project is configured for seamless deployment on **Vercel**. Simply push to a repository and connect it to Vercel.
